About

Это проект, который я создал основываясь на задании найденном на этом сайте. Делаю я с целью закрепить некоторые навыки, которыми я владел только в теории, проверить свои текущие навыки и выкатить какой-нибудь сайт, чтобы освоить deploy и написание Docker-файлов. Для меня оригинальная задача слишком легкая задача, по этому я придумал для себя усложняющие критерии.

Усложняющие критерии:

  1. Связь через gRPC
  2. Чат, который работает в реальном времени
  3. Лайки
  4. Комментарии
  5. Возможность писать блог с аккаунта администратора
  6. Весь код должен быть покрыт авто-тестами
  7. Задеплоить это всё на хостинг
  8. Добавить поддержку двух языков
